# User Documentation

## Overview

The Wellspring University Student Assistant helps students ask questions and receive quick guidance through a web chatbot.

## Pages

- `Home`: Introduces the assistant.
- `About`: Explains how the chatbot works.
- `Sign up`: Creates a student account with username, email, matric number, and password.
- `Login`: Signs in with matric number and password.
- `Reset Password`: Sends a password reset request.
- `Chatbot`: Allows signed-in students to ask questions and review previous conversations.

## Creating an Account

1. Open the signup page.
2. Enter username, email, matric number, and password.
3. Submit the form.
4. You will be redirected to the chatbot page.

## Logging In

1. Open the login page.
2. Enter matric number and password.
3. Submit the form.

## Asking Questions

1. Open the chatbot page after logging in.
2. Type a student support question.
3. The assistant returns the best answer and a confidence score.
4. Related questions appear when useful.

## Conversation History

The chatbot page shows saved conversations. These can help administrators review common student needs and improve the knowledge base.

# User Documentation

## Overview

The Wellspring University Student Assistant helps students ask questions and receive quick guidance through a web chatbot.

## Pages

- `Home`: Introduces the assistant.
- `About`: Explains how the chatbot works.
- `Sign up`: Creates a student account with username, email, matric number, and password.
- `Login`: Signs in with matric number and password.
- `Verify Email`: Confirms a new account from an emailed verification link.
- `Forgot Password`: Requests a reset link by email.
- `Reset Password`: Sets a new password from an emailed reset link.
- `Chatbot`: Allows signed-in students to ask questions.
- `History`: Shows saved chatbot conversations on a separate page.

## Creating an Account

1. Open the signup page.
2. Enter username, email, matric number, and password.
3. Submit the form.
4. Check your email and open the verification link.
5. After verification, log in or continue to the chatbot.

## Logging In

1. Open the login page.
2. Enter matric number and password.
3. Submit the form.
4. If your email is not verified, use the verification page to request another confirmation link.

## Resetting a Password

1. Open the forgot password page from the login form.
2. Enter the email linked to your account.
3. Open the reset link sent to your email.
4. Choose and confirm your new password.

## Asking Questions

1. Open the chatbot page after logging in.
2. Type a student support question.
3. The assistant returns the best answer and a confidence score.
4. Related questions appear when useful.

## Conversation History

The chatbot page shows saved conversations. These can help administrators review common student needs and improve the knowledge base.

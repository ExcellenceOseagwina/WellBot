About the system

Built for student assistance

The assistant is designed to help Wellspring University students find guidance quickly while preserving conversations that can improve support quality over time.

How the chatbot decides

When a student asks a question, the system tokenizes the text, compares it with stored university questions, calculates a confidence score, and returns the best answer. If confidence is low, it suggests related questions instead of pretending to know.

Scoring signals

Token overlap, keyword coverage, phrase matching, sequence similarity, and category matching.
